# **ZenithProxyForAndroid**
这是一个用于在Android设备上运行ZenithProxy的脚本

由于作者的一个朋友有此需求，所以作者写了这个脚本。

**！警告！：脚本下载的是3.4.2_java.1.21.4.pre版本的ZenithProxy核心，如需要更换为其他版本，请修改ZenithProxyForAndroid.sh中的第18行的wget命令**

============================================================================

## 如何使用？

**1.下载Termux**

2.在Termux中使用以下命令安装wget
```
pkg install wget
```
**3.执行以下指令以下载脚本**
```
wget "https://github.com/zychen027/ZenithProxyForAndroid/blob/main/ZenithProxyForAndroid.sh"
```
**4.执行以下命令以赋予脚本执行权限**
```
chmod 777 ".\ZenithProxyForAndroid.sh"
```
**5.使用以下命令以执行脚本**
```
.\ZenithProxyForAndroid.sh
```
**6.按照脚本提示使用**
